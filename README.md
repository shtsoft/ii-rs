A Rust workspace to develop a simple web-ready IRC client based on [ii](https://tools.suckless.org/ii).
The project consists of the packages:

- [iicli](iicli)
- [iifrontend](iifrontend)
- [iilib](iilib)
- [iiservice](iiservice)
